# Java Console Banking System

A console-based banking system built in Java demonstrating Object-Oriented Programming principles.

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- View transaction history

## Technologies Used
- Java
- OOP
- ArrayList

## Concepts Demonstrated
- Encapsulation
- Constructors
- Collections
- Loops & Switch statements
- Input validation
